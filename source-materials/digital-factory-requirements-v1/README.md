# Digital Factory Requirements v1

Canonical source materials for Factory Cloud / Digital Factory v0.

These files are reference inputs for specs, mock data contracts, API design, and workflow modeling.

## Files

- `Digital_Factory_Requirements.docx` — business requirements source document
- `Digital_Factory_Requirements.json` — structured requirements source
- `Digital_Factory_Requirements.md` — markdown version of requirements
- `Digital_Factory_Workflow.html` — workflow diagram package version
- `Digital_Factory_Flowchart.html` — standalone flowchart reference

## Usage

For implementation work, prefer this authority order:

1. `Digital_Factory_Requirements.json`
2. `Digital_Factory_Requirements.md`
3. `Digital_Factory_Workflow.html`
4. `Digital_Factory_Flowchart.html`
5. `Digital_Factory_Requirements.docx`

Do not edit these source files during implementation. Derived specs and app data should be created elsewhere.
